# Toronto Cafes Near Me

A Next.js cafe finder for discovering cute and cool cafes around Toronto.

## Live Website

Deployment pending. The Vercel URL will be added here after the production deployment is created.

## Features

- Nearby cafe discovery using browser location
- Toronto area presets
- Search, radius controls, and quick filters
- Ranked cafe cards with distance, tags, and directions links
- Public OpenStreetMap cafe lookup with saved Toronto fallback picks

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Privacy

This project does not require private API keys. Location is requested in the browser and is only used to fetch nearby cafe results.
